## Description

Profile for the Pelle cluster <https://www.uu.se/centrum/uppmax/resurser/kluster/pelle>.

115 nodes with AMD EPYC 9454P (Zen4) 2.75 GHz CPUs, 48 physical cores (96 threads), 768 GiB memory per node, and 1.7TB scratch area.

<https://docs.uppmax.uu.se/cluster_guides/slurm_on_pelle/#sbatch-and-interactive-on-pelle>
